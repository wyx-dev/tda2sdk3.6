calibration_tool.exe
====================
Tool to compute perspective calibration matrix for Surround View.

Inputs needed:
- 4 fisheye images with calibration charts
- reference image (top view)
- distortion_center.txt generated by fisheye_tool

Outputs:
- .c file containing perspective calibration matrix values
- .csv file containing perspective calibration matrix values

Requirements:
MCR (Matlab Compiler Runtime) 8.4
